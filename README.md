This repository contains my technical notes and hands-on lab documentation for the **CompTIA A+ 1200 series** certification. -----

## 📑 Study Areas

### **Core 1 (220-1201): Hardware & Networking**

  * **Mobile Devices:** Laptop hardware and mobile synchronisation.
  * **Networking:** IP addressing, ports, protocols, and SOHO configuration.
  * **Hardware:** Motherboards, CPUs, RAM types, and storage.
  * **Cloud & Virtualization:** Client-side virtualization and cloud service models.

### **Core 2 (220-1202): Software & Security**

  * **Operating Systems:** Windows 10/11, macOS, and Linux terminal basics.
  * **Security:** Malware removal, social engineering, and data destruction.
  * **Software Troubleshooting:** OS recovery and application stability.
  * **Operational Procedures:** Documentation, safety, and professionalism.
